# SimilarSets

## Bread basket dataset
Data needs to be preprocessed using `preprocBread.py` to produce `inBread` file.

Set :
```
filename = "inBread"
Support1 = 100
Support2 = 50
Support3 = 50
```

## Orders dataset
Data needs to be preprocessed using `preprocOrders.py` to produce `inOrders` file.

Set :
```
filename = "inOrders"
Support1 = 10k
Support2 = 8k
Support3 = 5k
```
